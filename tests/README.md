# ToDo list API
A API todo list that you can manage your to do list to your liking! Customized, add delete and manage all of your tasks, see which one is completed etc.
## Provided Features

    Add new item function
    Delete prior tasks
    Complete your list
    Wipe all completed tasks
    See all the completed or uncomplete items in your list

## Development Stack

Python, FASTAPI, Json (to store dict), VsCode IDE, Test PyPi
